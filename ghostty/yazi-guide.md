# yazi 使用指南

yazi 是现代化的终端文件管理器，支持预览和 Vim 风格操作。

## 🚀 启动

```bash
y              # 启动 yazi（退出时自动切换目录）
yazi           # 原生命令（不切换目录）
```

## ⌨️ 快捷键

### 导航
- `j/k` 或 `↓/↑` - 上下移动
- `h/l` 或 `←/→` - 进入/退出目录
- `g` - 跳到顶部
- `G` - 跳到底部

### 文件操作
- `Enter` - 打开文件/进入目录
- `o` - 用默认程序打开
- `O` - 交互式选择程序打开
- `y` - 复制（yank）
- `x` - 剪切
- `p` - 粘贴
- `d` - 删除
- `r` - 重命名
- `a` - 新建文件
- `A` - 新建目录

### 选择
- `Space` - 选择/取消选择
- `v` - 进入选择模式
- `V` - 取消所有选择
- `Ctrl + a` - 全选

### 搜索
- `/` - 搜索文件
- `n` - 下一个搜索结果
- `N` - 上一个搜索结果
- `z` - 使用 zoxide 智能跳转

### 其他
- `~` - 回到 HOME 目录
- `q` - 退出（并切换到当前目录）
- `Q` - 退出（不切换目录）
- `?` - 查看帮助

## 🔧 配置

配置文件：`~/.config/yazi/yazi.toml`

默认使用 Vim 作为编辑器（通过 `$EDITOR` 环境变量）。

## 💡 使用技巧

1. **快速跳转**：按 `z` 然后输入目录关键词
2. **批量操作**：`Space` 选择多个文件，然后执行操作
3. **预览文件**：自动预览文本、图片、PDF 等
4. **Shell 集成**：使用 `y` 命令启动，退出时自动切换目录
