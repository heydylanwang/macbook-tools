# Vim 使用指南

Vim 是强大的文本编辑器。

## 📦 安装

```bash
./install.sh
```

## ⌨️ 常用快捷键

### 模式切换
- `i` - 插入模式
- `Esc` - 返回普通模式
- `v` - 可视模式
- `:` - 命令模式

### 移动
- `h/j/k/l` - 左/下/上/右
- `w` - 下一个单词
- `b` - 上一个单词
- `0` - 行首
- `$` - 行尾
- `gg` - 文件开头
- `G` - 文件结尾

### 编辑
- `x` - 删除字符
- `dd` - 删除行
- `yy` - 复制行
- `p` - 粘贴
- `u` - 撤销
- `Ctrl + r` - 重做

### 保存退出
- `:w` - 保存
- `:q` - 退出
- `:wq` - 保存并退出
- `:q!` - 强制退出不保存

### 搜索
- `/关键词` - 向下搜索
- `?关键词` - 向上搜索
- `n` - 下一个
- `N` - 上一个

## 💡 使用技巧

- 普通模式下操作，插入模式下输入
- 组合命令：`d3w` 删除 3 个单词
- 重复命令：`.` 重复上次操作
